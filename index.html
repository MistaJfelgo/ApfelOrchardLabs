What this is

        
An orchard for surviving machines.

      

      
Apfel Orchard is the front gate for a set of personal computing preservation projects: the BBS, Wang Professional Computer recovery work, IBM clone notes, and future Amiga archives. Some branches are active. Some are still being grafted.
